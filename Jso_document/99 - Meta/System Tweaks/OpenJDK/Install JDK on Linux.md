> Check version 

```shell
sudo apt search openjdk-17
```

> Install Java JDK 

```shell
sudo apt install openjdk-17-jdk
```

> Verify 

```jsx
sudo java -version
```

> Install Java Extention Pack 

> Launch VS Code Quick Open (Ctrl+P), paste the following command, and press enter.

```jsx
ext install vscjava.vscode-java-pack
```
